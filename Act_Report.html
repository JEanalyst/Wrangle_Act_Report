Analysis and Insights¶

Four Insights were analysed and documented for this report, they are;

  • Most popular dog name
  • Most popular dog stage
  • Most popular dog breed
  • Correlation between the retweet count and favorite count

Most popular dog name¶

Analyzing this report comes with some limitation such as a large amount of unnamed dogs shown as none.

Using the value counts of name column i was able to get this analysis done, it shows the name "Cooper" as the most popular name, followed closely by "Charlie", "Oliver", "Koda' and "Sadie" while ignoring the none values which represents names with no records in our dataset.

Most popular dog breed¶

From my analysis, using the value counts recorded in our combined dataset the "Golden Retriever" as the most popular dog breed, having the highest value counts of 117, followed by "pembroke", "Labrado Retriever", "chihuahua' et al with 77, 72 and 50 respectively, please refer to the image below for more insight.
